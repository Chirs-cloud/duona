💜 多拿点 · 部署指南

只需要 3 步，AI 功能马上可用

1

Fork 项目到 GitHub

在 GitHub 新建账号（或登录）→ 点击 Fork 把这个项目复制到你的仓库

2

连接 Vercel 部署

打开 vercel.com → 用 GitHub 登录 → Import 项目 → 点 Deploy

Vercel 免费，不需要信用卡！

3

自动上线

部署完成后，你会得到一个 .vercel.app 链接，AI 功能自动生效！
